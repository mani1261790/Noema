# Notebook Layout

- `core/`: Noema original notebooks and non-NMA content.
- `nma/compneuro/`: Neuromatch Academy computational neuroscience notebooks.
- `nma/deep-learning/`: Neuromatch Academy deep learning notebooks.

Math authoring rules:

- Inline math: `$...$`
- Display math: `$$...$$`
- `\(...\)` and `\[...\]` are also supported, but for manual edits prefer dollar delimiters.
